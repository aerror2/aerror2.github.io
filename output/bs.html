这里只说silab的两个版本

1. 到Silicon Labs 注册帐号下载SimplicityStudio

2.到Product Downloads | Raisonance Extranet 下载Ride和RKit-51 

3. BLHeli目录下修改MakeHesfiles.bat

开头改成SilabsPath和 RaisonancePath你安装所在路径

SilabsPath=E:\SiliconLabs\SimplicityStudio\v5\developer\sdks\8051\v4.3.0\Device\C8051F330
SET RaisonancePath=E:\Raisonance

 结尾PIN参数改一下成实际路径：

%RaisonancePath%\Ride\bin\ma51.exe "BLHeli.asm" SET(BESCNO=%BESCNO%) OBJECT(Output\%BESCNAME%_%Revision%.OBJ) DEBUG EP QUIET PIN(%SilabsPath%\Inc;%RaisonancePath%\Ride\inc;%RaisonancePath%\Ride\inc\51) >> MakeHex_Result.txt

就可以双击编译BLHELI了。

3。 BLHELI_S目录下，修改一下MakeHesfiles.bat

开头，改成KeilPath你安装 的目录。

SET KeilPath=E:\SiliconLabs\SimplicityStudio\v5\developer\toolchains\keil_8051\9.60\BIN

就可以双击编译BLHELI_S了。
